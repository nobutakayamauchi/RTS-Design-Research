# Traceability Model

RTS Design Research deliverables connect reference queries, provenance sources, findings, UI decisions, and implementation briefs.

## Trace chain
1. `reference_queries` define the research intent.
2. `source_id` values identify each retrieved or approved reference.
3. Findings cite the evidence that influenced the design direction.
4. RTS design decision blocks preserve context, decision, constraints, assumptions, action, and outcome.
5. Implementation briefs translate the decision into buildable work.
