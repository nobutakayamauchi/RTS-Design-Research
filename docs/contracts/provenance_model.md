# Contract: Provenance Model

## Purpose
The provenance model standardizes source metadata captured during UI reference research.

## Standard source fields
- `source_id`
- `source_type`
- `provider`
- `query`
- `retrieved_at`
- `reference_url`
- `screenshot_path`
- `license_or_usage_note`
- `confidence`
- `notes`

Use `na` when a URL or screenshot path is unavailable. Do not drop the field.
