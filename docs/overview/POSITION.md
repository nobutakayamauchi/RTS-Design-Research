# POSITION

RTS Design Research is the design-intelligence extension layer for the RTS ecosystem.

It converts external UI references, screenshots, product patterns, and design research findings into reconstructable UI decisions.

## It is not
- the RTS trust core
- a runtime
- a raw MCP connector dump
- a generic design gallery
- a product implementation repository

## It is
- a design research contract layer
- a UI decision reconstruction layer
- a bridge from visual references to RTS-recordable design decisions
