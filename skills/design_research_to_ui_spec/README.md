# design_research_to_ui_spec

Converts UI pattern research into a report, UI spec, implementation brief, and RTS design decision block.

## Inputs
Required:
- `project`
- `screen`
- `research_goal`
- `reference_queries`

Optional:
- `constraints`

## Outputs
- `report_md`
- `ui_spec_md`
- `implementation_brief_md`
- `rts_design_decision_block_yaml`
