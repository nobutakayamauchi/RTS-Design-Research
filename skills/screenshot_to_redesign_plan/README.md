# screenshot_to_redesign_plan

Turns a screenshot and target direction into a redesign plan, UI spec, and RTS design decision block.

## Inputs
Required:
- `project`
- `screen`
- `screenshot`
- `target_direction`

Optional:
- `constraints`

## Outputs
- `redesign_plan_md`
- `ui_spec_md`
- `rts_design_decision_block_yaml`
